## Reproducing the analysis

[results/results.Rmd](results/results.Rmd) drives the analysis. When run, it sources R scripts from the [code/](code/) folder:

- `code/load_dependencies.R`
- `code/preprocess_data.R`
- `code/analysis_functions.R`
- `code/plotting_functions.R`
- `code/table_functions.R`

To reproduce: open `results/results.Rmd` in RStudio and knit, or from the project root run:

```r
rmarkdown::render("results/results.Rmd")
```

(Requires R and the `here` package; use the project root as working directory.)

Input data lives in [data/](data/).
